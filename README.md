## 🛠️ 技术栈说明
### 后端服务
- **核心框架**：Webman（PHP 高性能常驻内存框架）

  - 内置HTTP服务、WebSocket、定时任务，无需额外依赖
- **数据库**：MySQL 5.7+
- **缓存支持**：Redis（可选，用于提升接口性能）
- **接口对接**：原生支持淘宝联盟、多多进宝、京东联盟、美团联盟等主流CPS平台接口
- **架构优势**：模块化开发，代码结构清晰，注释完善，二次开发门槛低

- **适配终端**：默认微信小程序，一套代码可编译发布至H5、抖音小程序、支付宝小程序
- **UI 设计**：原生风格界面，交互流畅，适配主流机型，支持深色模式基础适配
- **工程化**：组件化开发，页面结构清晰，可快速自定义主题、替换logo、调整频道

## 🚀 部署说明
### 环境要求
- 服务器：Linux 系统（推荐 CentOS 7+/Ubuntu 20.04+），配置1核2G起步即可运行
- PHP 版本：≥ 7.4，需安装 pcntl、posix 扩展
- MySQL 版本：≥ 5.7
- 微信小程序账号 + 各电商平台联盟账号

### 部署步骤
1. 后端部署：上传 Webman 后端代码至服务器，配置数据库，启动服务
2. 后台配置：登录分销管理后台，配置各平台CPS接口密钥、基础设置、分销规则
3. 前端配置：修改 UniApp 前端接口地址、小程序AppID，编译上传至微信开发者工具
4. 上线发布：提交小程序审核，审核通过后正式运营

> 源码附带完整部署文档，步骤详细，有基础PHP环境搭建经验即可快速部署上线
